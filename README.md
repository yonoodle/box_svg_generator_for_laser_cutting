這個程式輸入盒子的長寬高以及厚度卡榫點的寬度作為參數 , 快速生成一個方便組合的盒子的SVG格式檔案,可以匯入各種編輯軟體做為後製修改各種孔洞以合乎應用上 ,

因年久遺失原始檔案,故只有執行檔 ,按下產生SVG按鈕之後就會在相同目錄下產生出SVG檔案 ,預覽功能暫時無法使用但不影響檔案生成.

This program takes the box's length, width, height, thickness, and tab width as input parameters to quickly generate an SVG file for an easy-to-assemble box. 
The file can be imported into various editing software for post-processing modifications such as adding holes to suit specific applications. 
Due to the loss of the original source files over time, only the executable file remains. After clicking the "Generate SVG" button, 
an SVG file will be created in the same directory. The preview function is temporarily unavailable, but this does not affect file generation.
